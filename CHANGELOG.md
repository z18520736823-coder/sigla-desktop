# Changelog

All notable public-preview changes to Sigla are documented here.

## [1.0.0-preview.1] - 2026-07-22

### Highlights

- Added a local, read-only Markdown review workspace for macOS.
- Added source-linked AI summaries powered by OpenAI or DeepSeek with BYOK.
- Added box annotations for comments, questions, and change requests.
- Added Agent-ready feedback with exact source locations and acceptance criteria.
- Added external-change detection, annotation re-anchoring, and review history.
- Added Codex return flow, Finder integration, and optional default Markdown opening.
- Added system, light, and dark themes with bilingual typography and Mermaid rendering.
- Added offline reading, annotation, and local fallback feedback.

### Distribution

- Universal 2 DMG for Apple Silicon and Intel Macs.
- Minimum macOS version: 11.0.
- Ad-hoc signed with hardened runtime; Apple notarization remains pending.
- Manual updates during the Public Preview.
