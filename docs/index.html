FAQ

Clear boundaries, before you download.

Is Sigla open source?

No. The public repository contains product documentation, community resources, and release downloads. The application source code remains private under a proprietary license.

Does Sigla edit my Markdown?

Sigla exposes no source operation for editing, deleting, moving, renaming, copying, restoring, or formatting Markdown. Annotations and review state are stored separately.

Can I use Sigla without AI?

Yes. Reading, rendering, annotation, file monitoring, and local basic feedback continue without an API key or network connection.

Which AI providers are supported?

Sigla 1.0 includes OpenAI and DeepSeek presets through a bring-your-own-key workflow. AI is optional and initiated by the user.

Does Sigla upload my documents?

Sigla has no document-sync server. An explicit AI action sends the necessary document content and review context directly to the provider you selected.
