# Welcome to the Sigla Public Preview

Sigla is the human review layer for AI-generated Markdown. It helps you turn long Agent output into source-linked insights, precise annotations, and actionable feedback while keeping the original file read-only.

[Download Sigla 1.0.0 Public Preview](https://github.com/z18520736823-coder/sigla-desktop/releases/tag/v1.0.0-preview.1) for macOS and tell us how it performs in a real Agent workflow.

We would especially love to hear:

- Whether AI conclusions and source links help you understand long documents faster.
- Whether box annotation and feedback handoff feel clear and lightweight.
- Where the bilingual reading experience or macOS integration can improve.
- Any installation issue, unexpected behavior, or missing capability.

Use [Bug reports](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=bug.yml), [Feature requests](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=feature.yml), or [Experience feedback](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=experience.yml) when you want a structured form. General discussion is welcome below.

Your original Markdown stays read-only. AI is optional and runs only when you explicitly choose OpenAI or DeepSeek with your own API key.

If Sigla makes your review workflow clearer, consider giving the repository a Star—it helps more Agent users discover the project.

# 欢迎体验 Sigla 公开预览版

Sigla 是面向 AI 生成 Markdown 的人类审阅层。它帮助你把冗长的 Agent 产出转化为带原文证据的洞察、精准批注和可执行反馈，同时让源文件始终保持只读。

[下载 Sigla 1.0.0 公开预览版](https://github.com/z18520736823-coder/sigla-desktop/releases/tag/v1.0.0-preview.1)，并告诉我们它在真实 Agent 工作流中的表现。

我们尤其希望了解：

- AI 结论与原文跳转能否帮助你更快理解长文档。
- 框选批注和反馈交接是否足够清晰、轻量。
- 双语阅读体验与 macOS 集成还有哪些提升空间。
- 安装问题、异常行为或仍然缺少的能力。

需要结构化提交时，可以使用 [Bug 报告](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=bug.yml)、[功能需求](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=feature.yml)或[体验反馈](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=experience.yml)。一般讨论可以直接回复本帖。

源 Markdown 始终只读。AI 是可选能力，仅在你主动选择 OpenAI 或 DeepSeek 并使用自己的 API Key 时运行。

如果 Sigla 让你的审阅流程更清晰，欢迎为仓库点亮 Star，帮助更多 Agent 用户发现它。
