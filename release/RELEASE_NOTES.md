# Sigla 1.0.0 Public Preview 1

**The human review layer for AI-generated Markdown.**

Sigla turns long, dense Agent output into evidence-linked insights, source-anchored annotations, and actionable feedback—while original files stay read-only.

## Highlights

- **AI-powered clarity:** OpenAI or DeepSeek turns long Markdown into one central conclusion and 3–5 source-linked takeaways.
- **Effortless annotation:** Draw a box around any passage and add a comment, question, or change request.
- **Agent-ready feedback:** Convert selected annotations into exact locations, requested changes, and acceptance criteria for Codex or another Agent.
- **Offline continuity:** Read, render, annotate, and generate local basic feedback without an API key or network connection.
- **Native macOS workflow:** Open Markdown from Finder or Codex, use system/light/dark themes, and run on Apple Silicon or Intel.

## Installation

1. Open the DMG and drag Sigla into Applications.
2. For the first launch, open Finder → Applications.
3. Control-click Sigla and choose Open.
4. If macOS still blocks the app, allow it in System Settings → Privacy & Security.

## Preview limitations

- The installer uses an ad-hoc signature with hardened runtime and has not completed Apple notarization.
- Updates are downloaded and installed manually.
- AI requires your own OpenAI or DeepSeek API key; all core reading and annotation features work without AI.
- The application source code remains private.
