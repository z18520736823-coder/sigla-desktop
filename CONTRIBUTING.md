# Contributing to Sigla · 参与 Sigla

Sigla's application source code is private. This public repository welcomes product feedback, reproducible bug reports, workflow ideas, documentation corrections, and safe security reports.

Sigla 应用源代码保持私有。本公开仓库欢迎产品体验反馈、可复现问题、工作流创意、文档修正和安全报告。

## Choose the right channel

- **Bug:** use the [bug form](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=bug.yml).
- **Feature request:** describe the review problem in the [feature form](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=feature.yml).
- **Experience feedback:** share where Sigla helped or slowed you down through the [experience form](https://github.com/z18520736823-coder/sigla-desktop/issues/new?template=experience.yml).
- **Question or idea:** start a [Discussion](https://github.com/z18520736823-coder/sigla-desktop/discussions).
- **Security issue:** follow [SECURITY.md](SECURITY.md) and use private vulnerability reporting.

## Protect your information

Before posting, remove:

- API keys and credentials
- Complete private paths and account names
- Confidential Markdown or feedback bundles
- Screenshots containing personal notifications or unrelated applications

## Documentation corrections

Small pull requests that correct public documentation, broken links, translations, or accessibility metadata are welcome. A pull request cannot include or request changes to private application source code.

Keep English and Chinese public claims equivalent. Product claims must match the behavior of the current downloadable build.
